### Create an HTML file

1. Open Brackets 
2. File &gt;&gt;&gt; Open folder
3. Select your local directory
4. File &gt;&gt;&gt; New
5. Paste in the following code for a basic HTML document
6. File &gt;&gt;&gt; Save

7. Save this new file as "index.html"

```
<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>Page Title</title>
    </head>
    <body>
        <!-- Add content here -->
    </body>
</html>
```
